# emotion-analyzer
Emotion analysis from text.
**dataset used for training the model**: https://huggingface.co/bulpara/emotion-analysis-8-categories


## To run the Frontend:
1. npm install
2. npm run dev

## To run the Backend:
1. pythom -m venv venv
2. ./venv/Scripts/activate.ps1
1. pip install -r "requirements.txt"
2. python app.py
